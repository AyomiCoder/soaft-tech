HTML images are defined with the img tag.

(src) means source, i.e this attribute imports your image when you include your image with the name you saved it with.

HTML Attributes

All HTML elements can have attributes
Attributes provide additional information about an element
Attributes are always specified in the start tag

List of Attributes

  • The href attribute: it can be applied to the a tag, and it provides address information for links
  • The src attribute: imports an image in your folder or Specifies the URL (web address) for an image
  • The width and hieght attribute: provides size information for images
  • The alt attribute: provides text for screen readers or displays as an alternate content if original content like image or video could not display properly
